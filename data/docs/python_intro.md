# Python 编程语言介绍

Python 是一种高级、解释型、通用型编程语言。由 Guido van Rossum 创建，于 1991 年首次发布。

## 主要特点

1. **简洁易读**：Python 的语法设计注重可读性，使用缩进来定义代码块。
2. **动态类型**：变量不需要声明类型，类型在运行时确定。
3. **面向对象**：支持类、继承、多态等面向对象特性。
4. **丰富的标准库**：自带大量实用模块，如 os、sys、json 等。
5. **跨平台**：可在 Windows、Linux、macOS 等多种操作系统上运行。

## 应用领域

- Web 开发（Django、Flask、FastAPI）
- 数据科学与机器学习（NumPy、Pandas、TensorFlow）
- 自动化脚本
- 人工智能
- 网络爬虫

## 版本信息

目前 Python 的主流版本是 Python 3.x 系列，Python 2 已于 2020 年停止维护。
